American Network Television Media

The table below demonstrates how often each of the three main network news agencies used a specific word (left column) within three minutes of using the word "Ukraine" compared to the average of all three agencies. A high value indicates a network used the selected word more in comparison to the other networks, while a lower value indcates a network used the selected word less in comparison to the other networks. Each value represents how many standard deviations each networks value is from the average of all three networks. All data is cumulative by year.



How to use this Chart

  • First, select the year from which you would like to view data.
  • The chart will display each network's relative useage for each word.
  • You can sort the chart for each network's values by clicking on the network name at the top of each column.
  • By clicking on the column header, you will sort the values for that column in either ascending or descending order on alternate clicks indicated by a small arrow to the left of each network name.
  • Red highlighted cells represent a higher value while blue highlighted cells represent a lower value.
